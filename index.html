O que é HTML?

VOLTAR AO INÍCIO DA PÁGINA

HTML (HyperText Markup Language, ou Linguagem de Marcação de Hipertexto) é a linguagem padrão usada para criar e estruturar
páginas na web. Ela define a estrutura e o conteúdo de um site, como textos, imagens, links, listas e outros elementos, por meio de tags
(ex.: <p>, <h1>, <ul>). As tags são instruções que o navegador interpreta para renderizar o conteúdo visualmente.

Como é uma estrutura básica de um HTML?

VOLTAR AO INÍCIO DA PÁGINA

A estrutura básica de um arquivo HTML, como o index.html, que é comumente usado como a página principal de um site, é composta
por três partes principais dentro da tag <html>: a tag <head> e a tag <body>, além da própria tag <html> que engloba tudo. Todas as
tags HTML são abertas com <nome-da-tag> e fechadas com "</nome-da-tag>".

Entendendo a diferença entre LISTAS ORDENADAS E NÃO ORDENADAS.

VOLTAR AO INÍCIO DA PÁGINA

No HTML existem 2 listas principais que são usadas para organizar o conteúdo de forma estruturadas. Estas listas se chamam: ordenadas e
não ordenadas.

    LISTA ORDENADA:

  • A tag <ol> define a lista ordenada, e cada item é marcado com a tag <li> (list item).
  • Por padrão os itens aparecem numerados, desta forma também é fácil de saber como é uma lista ordenada.

    LISTA NÃO ORDENADA:

  • A tag <ul> define a lista não ordenada, e cada item também usa a tag <li> (list item).
  • Por padrão os itens são marcados com "bullets points ." e seu estilo pode ser mudado com CSS.

O que é HEADER, MAIN, FOOTER?

VOLTAR AO INÍCIO DA PÁGINA

No HTML, as tags <main>, <header> e <footer> são elementos semânticos que ajudam a estruturar uma página web de forma clara, indicando o
propósito de cada seção. Elas melhoram a acessibilidade e a organização do código.

    HEADER:

  • Define o cabeçalho da página.
  • Geralmente contém elementos como logotipos, menus de navegação, títulos ou informações introdutórias.
  • Pode aparecer mais de uma vez em uma página, como no topo do site ou em seções internas.

    MAIN:

  • Representa o conteúdo principal da página, ou seja, a parte central e única do documento.
  • Deve haver apenas um <main> por página.
  • É usado para o conteúdo mais relevante, como artigos, listas ou formulários.

    FOOTER:

  • Representa o rodapé da página.
  • Costuma incluir informações como links de contato, direitos autorais, links para redes sociais ou créditos.
  • Pode aparecer mais de uma vez em uma página, como no final da página ou de um artigo.
